Algorithm:

0. Import header files/ modules.
     Python: import math
  C++:  #include<iostream>
        #include<cmath>

1. Get number of sides.
     Python: number = int(input("HOW MANY SIDES DOES POLYGON HAVE? "))
     C++:       cout << "HOW MANY SIDES DOES POLYGON HAVE? " ;
         cin >> number ;

2. Get side length.
     Python: length = float(input("length of a side: "))
     C++:        cout << "length of a side: " ;
          cin >> length ;

3. Area = number * (length * length) / (4 * tan(pi / number))
     Python: number*(length*length)/(4*math.tan(math.pi/number))
  C++:       number*(length*length)/(4*atan(pi/number)) ;

4. Display Area.
     Python: print ("THE AREA OF YOUR",number,"SIDED POLYGON IS ",area)
  C++:       cout << "THE AREA OF YOUR " << number ;
          cout << "SIDED POLYGON IS " << area ;
